description: Развертывание нового веб-проекта (scaffold)
---

Этот рабочий процесс помогает создать новое веб-приложение, следуя стандартам Antigravity.

1.  **Анализ требований**
    -   Убедитесь, что пользователь хочет создать именно веб-приложение (Vite/React и т.д.), а не просто HTML файл.
    -   Если нужен простой HTML, просто создайте файл.

2.  **Инициализация проекта**
    -   Используйте `npx` для создания проекта.
    -   // turbo
    -   Выполните команду создания (пример для Vite с React и SWC):
        ```bash
        npx -y create-vite@latest ./ --template react-swc
        # Затем установите Tailwind (если его нет в шаблоне, добавьте шаги инициализации)
        ```

3.  **Базовая настройка**
    -   Создайте/обновите `index.css`. Включите туда CSS Reset и базовые переменные (цвета, шрифты) согласно выбранной эстетике.
    -   Установите зависимости: `npm install`.

4.  **Разработка**
    -   Создавайте компоненты и страницы согласно задаче.
    -   Следуйте принципам "Frontend Design" (см. SKILL.md).
